# GameserverBot
Simple yet powerful bot that allows you to check your current players, map & more.

## Setup
1. Rename config.sample.json to config.json.
2. Edit it by putting in your information (bot token, MongoDB URI).
3. `npm install`
4. `npm start app.js`
